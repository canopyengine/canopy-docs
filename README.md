## Contributing

Documentation contributions are welcome.

You can help by:

- fixing typos
- improving explanations
- adding examples
- expanding guides
- clarifying architecture

To contribute:

1. Fork the repository
2. Create a branch
3. Submit a pull request

Clear and concise documentation helps make the engine easier to learn.
