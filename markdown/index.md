# Canopy Docs

**Welcome to the official documentation of ``Canopy Engine``, a 2D game engine built in ``Kotlin``.** This documentation is 
designed to provide you with all the information you need to:

* Use ``Canopy``
* Understand how it works

## Organization of the docs

This documentation is organized in the following sections:

* The [**Getting Started**](getting-started/index.md) section contains tutorials on how to get you started with ``Canopy``.

* The ``Manuals`` contain the main documentation on how to use the engine.

* The ``Engine details`` contain documentation about the engine itself, for those interested in contributing or learning 
about its inner workings.
